# Hantokana

### 简介

这是一款由python语言开发的用于将日文汉字转换为假名和罗马音的工具

假名（日语：**仮名／かな Kana **，日语发音：[kana]）为日本人创造的表音文字，与汉字共同构成日语的传统书写系统。

本应用不是中日互译词典，仅为拼写查询的小工具，所以没有很强的技术力，仅供娱乐Ciallo～(∠・ω＜)⌒☆

### 功能

支持 **汉字->平假名<->片假名->罗马音**

支持自定义词典

~~其实是汉字存在音读和训读两种情况，所以比较省事的办法是自己补充~~
